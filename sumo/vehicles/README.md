# Vehicle Images Directory

This directory contains custom vehicle images for SUMO simulation.

## Supported Formats
- PNG (recommended)
- GIF
- SVG

## Image Requirements
- Top-down view of vehicles
- Transparent background (PNG with alpha channel)
- Recommended size: 64x64 to 128x128 pixels
- Simple, clear car silhouettes work best

## Usage
The route file (`highway.rou.xml`) references images in this directory using the `imgFile` parameter.
